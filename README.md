## Dwarf Fortress like world Generation



**Key characters**
 - ≃ - Water
 - ⋛ - Sand
 - ↥ - Forest
 - ▲ - Mountain
 - ⋒ - Cities
 - \# - Cave Walls


**Camera controls**
 - W - Up
 - S - Down
 - D - Left
 - A - Right
 - Z - Elevation down
 - X - Elevation Up
 - E - EXIT
   


 **Installation**
 1. pip install -r requirements.txt
 2. open cmd/windows-terminal/terminal (windows-terminal better)
 3. Set width to 120 and height to 30 chars
 4. cd path/to/script/directory/
 5. python main.py
